1️⃣ What is the difference between var, let, and const?
>> var, let and const is used for variable declaration in JS.
var is function-scoped while let and const are block-scoped. 
var can be re-declared and updated. let can be updated but cant be re-declared. const can't be updated or re-declared.
var can be used before initializing and it declared as undefined. let and const can't be used before initializing and it gives an error. 

2️⃣ What is the spread operator (...)?
>> Spread operator is used to expand elements of an array or object.

3️⃣ What is the difference between map(), filter(), and forEach()?
>> map() is used to transform each elements of an array and create a new array. It does not change the original array.
filter() takes the elements that pass a condition and return them as a new array.
forEach() runs a function for each elements in an array. It doesn't return anything.

4️⃣ What is an arrow function?
>> The arrow function is a simpler version of function itself. If it is written in a single line it doesn't have to mention return.

5️⃣ What are template literals?
>> Template literals are used to create dynamic strings inside which variables can be inserted.
